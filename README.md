# Luhn-Algorithm-Project

这是一个用于学习 **Luhn 算法（模 10 算法）** 的 Python 示例项目。

## 📖 项目用途 (Purpose)
- **教育研究**：展示如何通过数学校验位验证数字序列的合法性。
- **前端开发**：用于测试网页表单输入框的正则匹配与逻辑校验。
- **软件测试**：在沙盒环境中模拟支付系统的 UI 响应。

## 🛠️ 使用说明
运行 `python generator.py` 即可在控制台输出符合算法格式的测试号码。
